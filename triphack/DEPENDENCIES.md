# Project dependencies

# Why I installed certain dependencies and their use cases

## Server Framework:
- **apollo server** - a server that allows us to host graphql queries and requests
- **express** - a backend server that allows us to mount REST apis
- **cors** - enables cross origin requests from the frontend

## GraphQL
- **graphql** - main dependency for graphql
- **graphql-tag** - support library for creating types

## Auth and Storage
- **Firebase** - used primarily for authentication of users and it's database, simple to install and straightforward to use with no costs 

## Styling
- **TailwindCSS** - used for styling the website, simple to use and easy to implement
